IOTA v1.4.003 - GUI Update

1. New set of controls that are automatically created from PHIL objects
2. Converted IOTA to use PHIL controls
3. Some GUI changes:
    - Moved "Run IOTA" button from toolbar down to lower left corner for better
      visibility to user
    - Changed the input list control to allow custom path addition, which reads
      wildcards
    - Combined most IOTA options under a single button
    - All Stills Processor options are now accessible as dialogs from IOTA
      interface; user can fiddle with them without editing the script
    - Image selection options are now present in the main IOTA window
    - A number of minor tweaks in IOTA and PRIME GUIs
